# Figma → React Prop Interface Inference

> Read Figma variant properties → map to TypeScript discriminated-union members → emit type-safe React props. Variant axes become discriminants, exhaustive switching is enforced by the type system.

## When to use
- Figma uses variant properties with enum-like values
- TypeScript strict mode
- Component prop interface is canonical

## When NOT to use
- Free-form variant labels
- JS-only project
- Props generated by another tool

## Glue summary
| Added element | Where |
|---|---|
| Discriminated-union emission (not flat prop bag) | Type generation |
| Variant-axis namespacing on collision | Pre-emit |
| Project type prefix convention enforcement | Pre-emit |
| AI-invented-axis guard | AI guard |
