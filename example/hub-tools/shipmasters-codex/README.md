# The Shipmaster's Polymorphic Codex

> **Why.** Compose a typed logbook of harbor commands with byte-aware beacon signals, exported as a polymorphic JSON manifest.

## Features

- Polymorphic command entities (Chart, Sail, Anchor, Signal, Lantern) with @type discriminator JSON export
- Byte-aware signal truncation preserves ellipsis within a 70-byte beacon budget
- Live manifest preview, route distance tally, and copy/download as cartographer-ready JSON
