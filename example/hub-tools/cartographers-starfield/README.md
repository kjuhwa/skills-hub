# The Cartographer's Forgotten Starfield

> **Why.** Draw forgotten constellations across a weathered starfield; browse and search with a bloom-filter visual catalog.

## Features

- Click stars to weave a constellation, name and save it to the weathered atlas
- Bloom-filter bit-grid visualizer for fast constellation name lookup
- Procedural drifting starfield with magnitude, hue, and shimmer

## File structure

```
cartographers-starfield/

  style.css     — dark-theme styling and animations
  app.js        — interactions, simulated data, render loop

```
