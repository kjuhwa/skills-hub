# Aurora Cartograph Atlas

> **Why.** Weathered cartographers watch dream-currents drift beneath shifting auroras and silver dunes in real time.

## Features

- Animated aurora bands and humming constellations using lantern-visualization-pattern (flicker envelopes + glow halos)
- Five named dream-currents rendered as time-series-db-visualization-pattern multi-series canvas lines
- Hover the canvas to probe the nearest current; drift/aurora sliders shift intensity; space toggles pause
