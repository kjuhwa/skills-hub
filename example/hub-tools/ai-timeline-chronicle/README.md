# AI Timeline Chronicle

> **Why.** 2023-2026 AI/Tech 마일스톤 57개를 수평 타임라인으로 시각화 — LLM, Agents, Open Source, Regulation, Hardware 5개 레인에서 흐름을 한눈에 파악

## Features

- Horizontal scrollable timeline spanning 2023–2026
- 5 category lanes: LLM, Agents, Open Source, Regulation, Hardware
- 57 event cards with date, title, summary, and 1–5 star impact rating
- Click card → expanded detail panel with full description
- Mouse drag, scroll wheel pan, Ctrl+wheel zoom, touch/pinch support
- "Surprise Me" button — random jump with smooth animation + glow flash
- Category filter pills to toggle lanes on/off
- Minimap for orientation
- Korean/English toggle for all content
- Animated particle field background
- Dark theme with category-specific accent colors
- Responsive (desktop + mobile)

## Usage

Open `index.html` in any modern browser. No server required.

Keyboard shortcuts: Arrow keys (pan), `+`/`-` (zoom), `S` (Surprise Me).
