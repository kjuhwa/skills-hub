# Silver Comet Voyage

> **Why.** Pilot a wandering cartographer across a grid sky, dodging silver comets while claiming stars into named constellations.

## Features

- Grid-based turn game: arrow keys move the cartographer one cell
- Procedural comet volleys threaten claimed stars with deterministic paths
- Stitch three adjacent claimed stars to name a constellation and score
