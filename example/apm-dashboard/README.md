# APM Suite Dashboard

> **Why.** Interactive APM tool with trace waterfall, metrics dashboard, and service topology map — showcases 7+ monitoring skills in a single zero-dependency app.

## Features

- **Trace Waterfall** — span hierarchy with color-coded services, click-to-inspect detail panel, error highlighting, 3 trace templates (GET users, POST orders, GET orders list)
- **Metrics Dashboard** — 4 golden signals (Latency P50/P95/P99, Throughput, Error Rate, Saturation) with real-time canvas charts, baseline band overlay, threshold alerting, frozen-metric detection
- **Service Topology** — interactive node graph (10 services, 12 edges), draggable nodes, animated traffic particles, health state simulation, node detail on click

## File structure

```
apm-dashboard/
  index.html        — Shell with tab navigation
  style.css         — Dark theme, layout, animations
  app.js            — All logic: trace gen, metrics sim, topology renderer

```

## Usage

