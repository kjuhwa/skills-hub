# Split-Brain Oracle Terminal

> **Why.** A green-on-black operator console that models quorum arithmetic, election latency, and partition recovery as pluggable knobs on a live D3 simulation.

## Features

- D3 force simulation of heartbeating nodes with `d3.forceSimulation`, live quorum rings, and partition cleaver — built around `consistent-hashing-visualization-pattern` and `raft-consensus-visualization-pattern`
- Scales/axes calculator deck for quorum, heartbeat budget, and worst-case election time driven by `d3.scaleLinear` and `d3.transition`, respecting `quorum-visualization-off-by-one` and `time-unit-consistency-us-ms-ns`
- Ledger of 30+ named cluster scenarios (dual-leader, minority-island, flapping-follower…) rendered with D3's enter/update/exit pattern and a terminal log inspired by `log-aggregation-visualization-pattern`
