# Origami Crane Constellation Atlas

> **Why.** Unfolds whispered paper cranes into drifting constellation maps above a lantern-lit tea garden canvas.

## Features

- Dual-layer canvas+SVG scene where cranes fly across a starfield and lanterns hum with incommensurate sine flicker
- Click any crane or lantern to inspect its "corpus item" (skill card) pulled from a seeded procedural inventory
- Ring-based consistent-hashing constellation map distributes cranes across nine tea-garden districts with hover tooltips
