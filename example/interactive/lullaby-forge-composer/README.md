# Lullaby Forge Composer

> **Why.** A deterministic forge that spins any seed phrase into a lullaby, melody roll, and portable binary packet.

## Features

- FNV-1a/xorshift seed pipeline turns a phrase into verses, syllable timings, and a pentatonic melody
- Strategy selector hot-swaps mood presets (silver-river, sleeping-fox, mountain-bell) with hysteresis preview
- Copy-to-clipboard exports: plain lyrics, SMS-byte-capped version, compact hex packet, and a melody timeline
