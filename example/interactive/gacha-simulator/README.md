# Gacha Simulator Arena

> **Why.** Interactive gacha/loot box simulator with soft/hard pity, character collection, and turn-based combat — showcases 3 game-system skills.

## Features

- **Gacha Pull System** — 3 banners, soft pity from pull 75 (+2%/pull), hard pity ceiling at 90, card flip animation with rarity glow
- **Character Collection** — 20+ characters across 4 rarities and 5 types, star-up via duplicates, stats panel
- **Turn-Based Combat** — pure-function state machine, SPD-based turn order, 4 actions (Attack/Skill/Defend/Item), wave progression
- **Status Effects** — Burn, Freeze, Poison, Shield, ATK/DEF Up/Down with turn tracking and visual indicators

## File structure

```
gacha-simulator/
  index.html        — 3-tab shell (Gacha/Collection/Combat)
  style.css         — Dark theme, card flip animations, rarity glows
  app.js            — All logic: pull system, collection, combat engine, effects
  manifest.json     — Hub metadata
```

## Usage

Open `index.html` directly in any browser. No server required.
