# Drifting Cranes Atlas

> **Why.** An animated silver-river atlas where paper cranes ferry forgotten lullabies past sleeping foxes and distant bells.

## Features

- Canvas scene with parallax mountain silhouettes, tide-rhythm river, and drifting origami cranes
- Click a crane to open its lullaby, reveal trace waterfall, and pulse its bell node on the topology map
- Side dashboard with crane queue, river health, and dream-replica lag — all deterministically seeded
