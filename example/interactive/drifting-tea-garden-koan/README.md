# Drifting Tea Garden Koan

> **Why.** A turn-based crane-folding puzzle where each fold kindles lanterns and compensates forgotten constellations.

## Features

- Pure-reducer combat-style turn engine where cranes cast folds against ember lanterns on a 7×7 tea-garden grid
- Soft/hard pity pulls fund a lantern deck, with status effects (burn, stun, dampen) modeled as enum-tagged records
- Saga-style compensation chain undoes a bad fold within the same turn if lantern humming falls silent
