# Clockwork Stitch Composer

> **Why.** A utility that composes constellation-stitch patterns from seed words, emitting coordinates, cadence, and hummingbird choreography.

## Features

- Seed-word hasher produces deterministic star graph, cadence tempo, and lantern schedule
- Exports stitch pattern as JSON / SVG / CSV using `llm-json-extraction` shape guards
- Live preview with copy-to-clipboard and keyboard-driven regeneration

## File structure

```
clockwork-stitch-composer/
  index.html    — shell, markup, inline SVG where used
  style.css     — dark-theme styling and animations
  app.js        — interactions, simulated data, render loop

```
