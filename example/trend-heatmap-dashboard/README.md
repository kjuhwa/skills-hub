# Trend Heatmap Dashboard

> **Why.** 36개 기술의 28개월간(2024.01–2026.04) 인기도 변화를 히트맵으로 시각화 — 어떤 트렌드가 지속되고 어떤 것이 사라졌는지 한눈에 파악

## Features

- Grid heatmap: 36 technologies × 28 months with heat gradient (navy → red)
- 6 category groups: AI/LLM, Frontend, Backend, DevOps, Database, Mobile
- Click cell → detail modal with sparkline, score, trend badge, context note
- Rising Stars sidebar: top 8 by 6-month growth slope
- Fading sidebar: bottom 6 declining technologies
- Sort by: Category, Name, Current Score, Growth Rate
- Search/filter bar for instant row filtering
- Stats bar: total techs, average score, rising/declining counts, hottest tech
- Korean/English toggle for all content
- Dark theme with smooth hover effects and row highlighting
- Responsive layout
