Parallel Dispatch Coverage Gate — Benchmark

Experiment artifact for paper workflow/parallel-dispatch-breakeven-point. Tests whether a ~70% coverage threshold for "parallel stops paying off" holds across multiple measured corpus domains.
Premise being tested: IF a bulk-modification task is dispatched to N parallel subagents without a pre-flight coverage check, THEN beyond a prior-work coverage threshold of roughly 70%, the parallel pattern becomes a net negative.
