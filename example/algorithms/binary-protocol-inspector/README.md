# Binary Protocol Inspector

> **Why.** Hex editor + binary protocol decoder with custom schema designer, VarInt visualizer, and field-to-byte highlighting — showcases wire protocol and frame envelope skills.

## Features

- **Hex Editor** — 16-byte rows with offset/hex/ASCII columns, click-to-select, keyboard editing, Ctrl+G/Ctrl+F shortcuts
- **Protocol Schema Designer** — visual field definition (18 types), drag reorder, 5 preset schemas (Simple Header, MQTT-like, WebSocket Frame, Protobuf-like, Custom TLV)
- **VarInt Visualizer** — per-byte continuation/data bit display, unsigned + ZigZag signed encoding, step-by-step decode
- **Decode/Encode View** — schema-applied tree view with color-coded field regions, hover cross-highlighting, encode mode for generating bytes from field values

## File structure

```
binary-protocol-inspector/
  index.html        — Shell with hex grid, schema panel, decode tree
  style.css         — Dark monospace theme, field color system
  app.js            — All logic: hex editor, schema engine, decoder, VarInt
  manifest.json     — Hub metadata
```

## Usage

Open `index.html` directly in any browser. Loads with Simple Header preset + sample data decoded.
