# Meridian Partition Oracle

> **Why.** Compute token ranges, replication budgets, and rebalance migration cost as a force-directed terminal for capacity planning.

## Features

- Force-directed ring of token owners and candidate layout overlay for what-if rebalance planning with live d3 transitions
- Replication-factor calculator: estimates durability, quorum read/write math, and data-transfer cost for an N→M node migration
- Scrolling TSV-style terminal output for reproducible capacity plans and a token range bar chart with enter/update/exit joins
