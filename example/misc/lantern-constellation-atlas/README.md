# Lantern Constellation Atlas

> **Why.** Explore a living night sky where lanterns drift as stars, each carrying a wish from a faraway festival.

## Features

- Animated canvas sky with drifting lantern-stars and twinkling parallax
- Click any lantern to read its wish and see connecting constellation lines
- Filter by festival region to reshape the constellation in real time
