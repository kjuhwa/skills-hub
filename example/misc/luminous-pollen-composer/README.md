# Luminous Pollen Composer

> **Why.** Seed-driven generator that composes apiary blueprints, winding cadence, and pollen flow manifests from any phrase.

## Features

- Type any phrase and emit a deterministic apiary manifest with hive roster, cadence schedule, and pollen routing table
- Live preview table, JSON export, and CSV cadence export with copy-to-clipboard
- Tiered rebalance presets (fast/mid/slow) and risk-gate guard rails annotate every hive entry
