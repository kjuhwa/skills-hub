# Watermark Offset Calculator

> **Why.** Compute consumer lag projections and checkpoint intervals from ingestion rate, processing throughput, and partition count inputs.

## Features

- Input sliders for produce rate, consume rate, partitions, and commit interval
- D3 stacked area chart projecting lag growth over configurable time horizon
- Break-even throughput indicator with animated threshold line
- Checkpoint frequency optimizer suggesting commit cadence for target lag
- Exportable JSON report of computed offsets, watermarks, and rebalance cost
