# Drifting Ember Flight

> **Why.** Guide a paper lantern through a stormy sky, balancing heat and wind to reach the distant shrine.

## Features

- Physics-driven flight where you feed the flame to climb and glide
- Procedural gust zones and obstacles that test timing and heat management
- Distance tracker with personal-best memory and a restart shortcut
