# Whispering Lantern Keep

> **Why.** A turn-based puzzle where you light lanterns to contain shadow-tides creeping through a haunted keep.

## Features

- 10×10 grid with adjacency lighting rules and shadow propagation each turn
- Three lantern types (pulse, warden, beacon) with distinct radius/decay
- Win/lose detection, undo stack, move counter, deterministic seed puzzles
