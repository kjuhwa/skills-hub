# Ember Token Foundry

> **Why.** Forge virtual tokens, rebalance shards, and defend a molten hash ring in a turn-based cluster-smithing simulation game.

## Features

- Turn-based ring-forge where you mint virtual tokens, weld nodes onto the ring, and resolve migration orders with bounded budgets
- Random incidents (node-leave, zone failure, replica lag spike, hot range) triggered each tick with chip-based mitigation cards
- SVG-rendered ember ring with copper-patina gradients, crisis log, and score based on balance entropy and migration cost

## File structure
