# Clockwork Apiary Atlas

> **Why.** Interactive dusk-lit canvas atlas of brass hives with drifting luminous pollen trails and copper wheat silhouettes.

## Features

- Seeded procedural hive placement with copper wheat parallax and dusk-to-night time slider
- Hoverable brass hives expose swarm metrics, pollen throughput, and gear-winding tension in a side panel
- Animated pollen particles drift along foraging arcs between hives with organic flicker
