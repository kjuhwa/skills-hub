# Partition Lag Observatory

> **Why.** Explore consumer group offsets and broker watermarks through a live multi-partition dashboard revealing lag hotspots instantly.

## Features

- Animated partition lanes showing log-end offset vs committed offset gap
- Consumer group selector with per-member assignment highlighting
- Hover tooltips revealing lag delta, throughput, and checkpoint timestamps
- Rebalance event timeline overlay with partition reassignment arrows
- Color-coded lag severity heatmap with threshold filter slider
