# Tech Trends Radar 2025-2026

> **Why.** ThoughtWorks식 인터랙티브 기술 레이더 — 43개 기술의 Adopt/Trial/Assess/Hold 포지셔닝을 한눈에

## Features

- ThoughtWorks-style SVG radar with 4 concentric rings (Adopt/Trial/Assess/Hold)
- 4 quadrants: Languages & Frameworks, Platforms, Tools, Techniques
- 43 interactive blips with click-to-detail popup
- Trend indicators per technology (↑ rising, → stable, ↓ declining)
- Sidebar filters by quadrant and ring with live counts
- Korean/English toggle for all UI labels and descriptions
- Dark theme with vibrant quadrant-specific accent colors
- Responsive layout (desktop + mobile)
